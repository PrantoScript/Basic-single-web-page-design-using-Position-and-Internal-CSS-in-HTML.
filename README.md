# Basic Single Web Page Design

This project demonstrates a basic single web page design using HTML and Internal CSS. The webpage is designed with different CSS position properties to create a structured and visually appealing layout.

## Features
- Single Web Page Design
- Internal CSS Styling
- CSS Position Properties
- Simple and Clean Layout
- Navigation Menu
- Header and Footer Section

## Technologies Used
- HTML5
- CSS3 (Internal CSS)

## Purpose
The main objective of this project is to understand the use of Internal CSS and CSS Position properties in web page design.

## Position Properties Used
- Relative Position
- Absolute Position
- Fixed Position

## How to Run
1. Download or clone the project.
2. Open the `index.html` file in any web browser.
3. Explore the webpage design.
